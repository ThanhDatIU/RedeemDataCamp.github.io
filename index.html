About DataCamp

Companies and teams of every size use DataCamp to close their data skill gaps and make better data-driven decisions.

DataCamp provides an online IDE, which can run the code directly in (with step-by-step tutorials). It has many free courses like Introduction to the Tidyverse, Introduction to SQL, Introduction to Data Science, but if you want to be more technical, you have to buy its subscription to have access to its full course.

DataCamp has a lot of courses on R, which if you search Google or find on other online teaching sites, there are not so many and the quality is not as good as DataCamp also. DataCamp, as its name clearly states, only focuses on Data Science, updates with the latest, constantly trending so that the skills is not out of fashion.

GitHub Student Developer Pack

#1: Get a GitHub account registered by email provided by IU (Office 365) then visit
GitHub Student Developer Pack and click Get your pack.

#2: Proceed Get student benefits.

#3: In What e-mail address do you use for school? Here I'm using my personal gmail, it is recognized as the right university format though. I suggest you should use your school email for faster approval.

#4: In What is the name of your school? Here I type in International University VNU and click on the suggested International University - VNU - HCM.

#5: In How do you plan to use GitHub? I use GitHub for educational purposes is a simple reply you could try. Then click Continue.

#6: In Proof Type specify Dated school ID.

#7: In Upload an image upload your IU ID.

#8: Process my application

#9: Receive an e-mail from GitHub that confirms your application. You shall need to wait for 2 weeks before moving to the last step.

#10: Visit GitHub Student Developer Offers and redeem DataCamp.
